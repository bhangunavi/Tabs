# Tab Functionality Project

A simple JavaScript project demonstrating **DOM manipulation** and **event handling** by building an interactive tab system.

## Features

- 4 clickable tabs
- First tab active by default
- Switches visible content on tab click
- Uses vanilla JavaScript (no libraries)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```bash
project-folder/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How It Works

1. The page loads with **Tab 1 active** by default.
2. Clicking another tab:
   - removes active state from current tab
   - hides current content
   - shows selected tab content

## JavaScript Concepts Covered

- `querySelectorAll()`
- `addEventListener()`
- `forEach()`
- `classList.add()`
- `classList.remove()`
- `dataset`

## Setup Instructions
